\# AI Weather Prediction System



\## Overview

This project is a weather monitoring and forecasting system built for CEME.  

It combines frontend, backend, database, and AI forecasting ideas into one system.



\## Features

\- Live weather data display

\- Forecast section

\- Historical weather viewing

\- Export data to CSV

\- Backend API using FastAPI

\- Frontend dashboard for users



\## Tech Stack

\- Python

\- FastAPI

\- Uvicorn

\- SQLite

\- JavaScript

\- HTML

\- CSS

\- ESP32 integration concept



\## Project Structure

\- `backend/` → backend code and API

\- `frontend/` → user interface files

\- `data/` → stored forecast or data files
